# Distributed Job Scheduler

A web-based **Distributed Job Scheduler** built with Python and Flask that allows users to submit, schedule, monitor, and manage jobs through an interactive dashboard.

## Features

- User-friendly web dashboard
- Create and schedule jobs
- Monitor job status in real time
- Job execution history
- SQLite database integration
- REST API support
- Responsive web interface
- Project documentation included

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Project Overview

This project demonstrates the implementation of a distributed job scheduling system. Users can create and monitor scheduled jobs through an interactive web interface. The application stores job information in a SQLite database and provides an efficient way to manage and track job execution.
